# FACTORIO-DEDICATED-SERVER-TUI
FACTORIO DEDICATED SERVER TUI é um projeto de interface em modo texto (TUI/CLI) criado para automatizar servidores dedicados de Factorio de forma simples, rápida e confiável em ambientes Windows.  O projeto foi desenvolvido com foco em administração de servidores, eliminando tarefas repetitivas e reduzindo falhas humanas ao iniciar.
